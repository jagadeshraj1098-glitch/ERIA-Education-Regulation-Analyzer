# 📘 ERIA — Education Regulation Impact Analyzer

### GUVI × HCL Capstone Project

## 🔗 Live Demo
[Open App](https://huggingface.co/spaces/jagadesh1098/ERIA)

## 📌 Problem Statement
Education policies from UGC, AICTE, and NAAC are often difficult to interpret.
ERIA converts them into clear, stakeholder-focused summaries.

## ✨ Features
- Upload regulation PDFs
- AI-powered plain-English summaries
- Stakeholder impact analysis
- Timeline impact assessment
- Positives / Risks breakdown

## 🛠️ Tech Stack
- Python
- Streamlit
- Groq API
- Llama 3.3 70B
- Hugging Face Spaces
